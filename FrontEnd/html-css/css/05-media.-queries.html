Design responsive

Il design responsive è un approccio alla progettazione di siti web che mira a creare pagine web che si adattano automaticamente alle dimensioni dello schermo e al dispositivo dell'utente. Questo significa che un sito web può essere visualizzato correttamente su una vasta gamma di dispositivi, dai desktop ai tablet fino agli smartphone.

Per creare un design responsive, gli elementi della pagina (testi, immagini, contenuti in generale) devono ridimensionarsi e riorganizzarsi in base alla larghezza dello schermo
Questo permette anche di avere un solo sito web da gestire per tutte le piattaforme, riducendo il bisogno di versioni separate per ogni tipo di dispositivo

Mobile first

Il mobile first è un approccio che propritizza la creazione di esperienze ottimali per dispositivi mobili prima di adattare a schermi più grandi, come desktop e tablet.

Si inizia a progettare il sito per i dispositivi di dimensioni più piccole assicurandosi che tutte le funzionalità siano accessibili e utilizzabili su smartphone.
Dopo aver creato una base solida per dispositivi mobili, si aggiungono funzionalità è miglioramenti per schermi più grandi, utilizzando le media queries.

Media Queries

Le media queries sono una funzionalità del CSS che consente di applicare stili specifici a un documento HTML in base alle caratteristiche del dispositivo su cui viene visualizzato, come la larghezza dello schermo, l'orientamento e la risoluzione. Sono fondamentali per il design responsive e per garantire che un sito web si adatti a diverse dimensioni di schermo.
