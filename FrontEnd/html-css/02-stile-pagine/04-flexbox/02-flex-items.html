Flexbox

Flex items

Una volta che abbiamo creato un container flex e gli abbiamo assegnato tutte le proprietà che ci servono per ottenere il layout desiderato, possiamo agire sui figli, quindi sui flex items, tramite altre proprietà CSS dedicate.

Order

Serve a controllare l'ordine in cui gli elementi flex vengono visualizzati all'interno di un contenitore flex. Questa proprietà è utile quando vogliamo modificare l'ordine degli elementi senza cambiare la loro posizione nel markup HTML.

flex-grow

Questa proprietà ci permette di far espandere gli elementi di un container flex per tutto lo spazio disponibile e di determinare quanto ogni singolo elemento debba crescere rispetto agli altri fratelli. (DEFAULT = 0)

1
2
3

Flex-shrink

Al contrario di grow, definisce quanto un elemento flex debba ridursi rispetto agli altri fratelli/elementi. Più grande il valore, più l'elemento viene ridotto. (DEFAULT = 1)

1
2
3

Flex-basis

Definisce la dimensione iniziale di un elemento flex prima che lo spazio disponibile venga distribuito secondo le proprietà [flex-grow] e [flex-shrink]. In altre parole, flex-basis determina la dimensione di un elemento lungo l'asse principale del contenitore flex.

Align-self

Consente di specificare l'allineamento di un singolo elemento all'interno di un contenitore flex lungo l'asse trasversale. Questa proprietà sovrascrive il valore di align-items impostato sul contenitore, permettendo un controllo più fine sull'allineamento degli elementi.
